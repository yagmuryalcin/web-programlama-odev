# Web Programlama Ödevi

Bu proje, ASP.NET Core MVC ve MS SQL Server kullanılarak geliştirilmiş bir
**Proje ve Görev Takip Sistemi**dir.

## Kullanılan Teknolojiler
- ASP.NET Core MVC (.NET 8)
- Entity Framework Core
- ASP.NET Identity
- MS SQL Server
- Git & GitHub

## Özellikler
- Kullanıcı kayıt / giriş (Identity)
- Proje oluşturma ve yönetme
- Görev ekleme, atama ve durum takibi
- Yetkilendirme (User bazlı)

> Bu proje, Web Programlama dersi kapsamında geliştirilmiştir.
